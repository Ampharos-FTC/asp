Poses

A Pose is the robot position inside of the field, it has X, Y and Heading, for simplicity X and Y are measured in Inches and Heading is in Degrees.

Every pose in ASP needs 3 values, Robot X, Robot Y, and Robot Heading. Creating a Pose with those values is very easy : Pose2D myPose1 = new Pose2D(67, 41, 5); Pose2D myPose2 = new Pose2D(67, 41, 5);


Paths

Paths are used by our MecanumDriveTrain
A Path in ASP consits of many Poses.

Making a new Path is also very easy. Pose2D myPose1 = new Pose2D(67, 41, 5); Pose2D myPose2 = new Pose2D(41, 67, -5); Path myPath = new Path() .add(myPose1) .add(myPose2);
